| 컴포넌트 | 역할 |
|----------|------|
| Frontend | Next.js 16 App Router, Zustand 상태관리, 클러스터 시각화 |
| Backend | FastAPI REST API, 비즈니스 로직 |
| aerospike-py (내장) | Backend에서 Aerospike 데이터 접근 (AsyncClient) |
| K8s Client | ACKO CRD 조회, Pod 상태 모니터링 |
| Database | 연결 정보, 사용자 설정 저장 |
