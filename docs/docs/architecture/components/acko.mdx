# ACKO Component Diagram

| 컴포넌트 | 역할 |
|----------|------|
| AerospikeCluster CRD | 원하는 클러스터 상태를 선언적으로 정의 |
| Admission Webhook | CE 제약 사전 검증 (security, TLS, XDR 차단) |
| Reconciler | CR ↔ 실제 상태 수렴 (StatefulSet, Service, ConfigMap, PVC 관리) |
| Aerospike Pod | Rack-aware 배포, metrics exporter 내장 |
| Helm Charts | Operator + CRD 설치 자동화 |
