|----------|------|
| Python API Layer | `Client` (동기) / `AsyncClient` (비동기) 인터페이스 제공 |

| aerospike-client-rust v2 | 핵심 Aerospike 프로토콜 구현 (연결 관리, 클러스터 디스커버리, 배치 처리) |
