| 계층 | 역할 |
|------|------|
| Python API | `Client` (동기) / `AsyncClient` (비동기) + CDT, Expression, Query |
| PyO3 Bridge | Rust ↔ Python 바인딩, GIL-free 비동기 지원 |
| Rust Core | wire protocol, connection pool, backpressure, OpenTelemetry |
| 연결 대상 | ACKO 관리 K8s Pod 또는 Bare Metal 서버 모두 지원 |
